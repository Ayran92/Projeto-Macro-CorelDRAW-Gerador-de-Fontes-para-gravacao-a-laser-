# Projeto-Macro-CorelDRAW-Gerador-de-Fontes-para-gravacao-a-laser-
Este projeto contém uma macro desenvolvida para o CorelDRAW, capaz de gerar automaticamente múltiplas variações de texto utilizando diferentes fontes, tamanhos e alinhamentos. A macro organiza os textos em colunas, aplica margens personalizadas, permite quebra de linhas via caractere |, numera cada bloco e posiciona tudo de forma precisa na página.
Foi criada para agilizar a visualização e comparação de fontes no processo de criação gráfica.

Principais recursos:

Geração automática de textos em 14 fontes diferentes.

Escolha de alinhamento (esquerda, centro ou direita).

Suporte a múltiplas linhas usando |.

Ajustes de margens, espaçamentos e posicionamento dinâmico.

Organização em duas colunas conforme o layout da página.
